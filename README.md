# 健身房管理系统 java38

## 项目概述

本项目是基于Java和SpringBoot框架，结合Vue实现的前后端分离的健身房管理系统。系统提供了完整的源码注释，便于开发者和使用者深入理解和二次开发。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 支持Windows和Mac操作系统

### 开发环境
- IDE：IDEA
- JDK版本：1.8
- 构建工具：Maven
- 数据库：Mysql

### 技术框架
- 后端：Java、SpringBoot、Mybatis
- 前端：Vue
- 数据库：Mysql 5.7

## 功能模块

### 用户系统
- 用户登录

### 管理员功能
- 会员管理
- 健身房管理
  - 课程管理
  - 器材管理
- 员工管理
- 报名表管理

### 会员功能
- 课程查询
- 在线充值
- 充值记录
- Vip管理
- 购买记录
- 我的资料
- 修改密码

## 系统角色
- 系统管理员：负责系统维护和用户管理。
- 员工：负责日常业务操作，如课程安排、器材管理等。
- 会员：使用系统提供的各项服务，如课程查询、在线充值等。

## 运行环境

本项目可在以下环境中运行：
- 后端：JDK 1.8及以上版本，Mysql数据库5.7版本
- 前端：现代浏览器

## 部署步骤

1. 准备开发环境：安装IDEA、JDK 1.8、Mysql、Maven等。
2. 克隆或下载项目代码。
3. 导入数据库脚本。
4. 配置项目中的数据库连接。
5. 编译并运行后端服务。
6. 前端项目使用Vue CLI或其他工具运行。
